# Yes Chef !

A modular cooking and kitchen-management game prototype built in Unity, focused on scalable interaction systems, order management, and extensible gameplay architecture.

---

The project emphasizes:

- Modular interaction systems
- Order generation and management
- Reusable kitchen object workflows
- Scalable gameplay architecture
- Extensible controller-driven design
- Custom gameplay logic built with Unity and C#

---

## Tech Stack

- Unity Engine
- C#
- ShaderLab
- HLSL

---

## Project Structure

```text
Assets/
├── Scripts
├── Art
├── Materials
├── Prefabs
├── Scenes
└── Shaders

Packages/
ProjectSettings/
```
---
## How To Play:
- Player Movement : WASD
- Interaction (Any Kind) : E
- Mouse : Drag and Drop Items
- Pause Game: Esc
